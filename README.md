# MartianLee's Dev Blog

개인 개발 기록과 지식 아카이브를 위한 Next.js 기반 블로그입니다.

- Blog: 기술 글/회고
- KB: 짧은 노트와 정리
- Projects: 개인 프로젝트 소개
- About: 경력 및 소개

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Contentlayer2 + MDX
- TypeScript

## Local Development

### Requirements

- Node.js `>= 24`
- Yarn `3.x`

### Install

```bash
yarn install
```

### Run

```bash
yarn dev
```

기본 개발 서버 포트는 `3456`입니다.

## Build

```bash
yarn build
yarn serve
```

## Content

- Posts: `data/posts`
- Authors: `data/authors`
